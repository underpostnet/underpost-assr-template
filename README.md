## UNDERpost.net | API and Server Side Render Template

![performance](https://github.com/underpostnet/underpost-assr-template/blob/master/performance/1.png?raw=true)

### Usage:


- install:


`npm install`


- update:


`npm run update`


- run dev mode:


`npm run dev`


- run mocha unit test:


`npm test`


- run production mode:


`npm start`


- reset modules:


`npm run reset`


- create docker image


`docker build . -t underpost-assr-template`
